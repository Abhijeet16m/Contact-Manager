# Contact Manager (Python)

A simple command-line Contact Manager built using Python and JSON for data storage.  
This project allows users to securely manage and maintain their contact list.

## Features
- First-time login setup
- Add new contacts
- Delete existing contacts
- View all saved contacts
- Search contacts by name
- View your own profile details

## Technologies Used
- Python 3
- JSON for file-based data storage

## How to Run
1. Ensure Python 3 or later is installed.
2. Download or clone this repository.
3. Open the terminal inside the project folder.
4. Run the code

## Notes
- Login details are created on first use and stored locally.
- Data persists even after the program is closed.
- This project is beginner-friendly and can be extended further.
